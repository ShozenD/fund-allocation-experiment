mainGraph

Displays a donut graph for the subjects budget allocation. Requires a predefined HTML element with element id set to "my-alloc-graph".

mainGraph(alloc: Object)
Parameters
alloc (Object) the subject's budget allocation(an instance of the Allocation Class)

comparisonGraph

Draws donut graphs that compare two different budget allocations. The x, y parameters are optimized for the group allocation stage.

comparisonGraph(el: HTMLElement, own: Object, other: Object, update: Boolean)
Parameters
el (HTMLElement) The HTML element in which to attach the graph to.
own (Object) The subject's own budget allocation(an instance of the Allocation Class).
other (Object) The budget allocation to compare to(an instance of the Allocation Class).
update (Boolean = false) Changes the behavior of the function for updating the comparison graphs

voteComparisonGraph
