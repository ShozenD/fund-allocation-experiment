Brief Introduction

This JavaScript Library was created to simplify the coding process on LIONESS Lab. It contains two files, Allocation.js, Table.js, and Timer.js. Although some stages may not require the import of all files, it is convienient to have all files imported. They also require the import of the BootStrap CSS Library inorder to render desirably. Thus it is recomended that the following code is included within the first E-1 text box of every stage. 

          
      <head>
        <!-- Required meta tags -->
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
              
        <!-- Bootstrap CSS -->
        <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css"
        integrity="sha384-ggOyR0iXCbMQv3Xipma34MD+dH/1fQ784/j6cY/iJTQUOhcWr7x9JvoRxT2MZw1T" crossorigin="anonymous">

Timer.js

Experiment Timer

There is a 30 minute time limit for the group allocation stage, and therefore a timer was needed to keep track of remaining time. Because the group allocation stage spans across several windows, a client side JavaScript function called sessionStorage was utilized, to keep track of the deadline across different windows(this function is similar but faster than using cookies). Once the timer is initialized, it will log the deadline in the sessionStorage and alert the player 15, 10, 5, and 1 minute before timeout.

How to use

First, include the following script element in first E-1 text box of the page.

            
        <head>
          <script src="https://shozend.github.io/fund-allocation-experiment/src/Timer.js"></script>
        <head>

To start the timer, simply call the initializeClock() function. initializeClock() takes id argument. Pass to it the id of the html element in which you want to render your clock. In this case the html element has an id of 'clockdiv', thus we pass it to the initializeClock() function. Within the html element include two span elements with class="minutes" and class="seconds" shown below 

              
        <!-- html -->
        <div id="clockdiv">
          Remaining Time: <span class="minutes"></span>: <span class="seconds"></span>
        <div>    

        <!-- JavaScript -->
        initializeClock('clockdiv')
