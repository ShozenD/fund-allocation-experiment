Brief Introduction

This JavaScript Library was created to simplify the coding process on LIONESS Lab. It contains two files, Allocation.js, and Table.js. Although some stages may not require the import of both files, it is still recomended that both files are imported. They also require the import of the BootStrap CSS Library inorder to render desirably. Thus it is recomended that the following code is included within the first E-1 text box of every stage. 

          
      <head>
        <!-- Required meta tags -->
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
              
        <!-- Bootstrap CSS -->
        <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css"
        integrity="sha384-ggOyR0iXCbMQv3Xipma34MD+dH/1fQ784/j6cY/iJTQUOhcWr7x9JvoRxT2MZw1T" crossorigin="anonymous">

        <!-- Additional JavaScript -->
        <script src="https://shozend.github.io/fund-allocation-experiment/src/Allocation.js"></script>
        <script src="https://shozend.github.io/fund-allocation-experiment/src/Table.js"></script>
      <head>
